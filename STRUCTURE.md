# Repository Structure

```
abstraction-intelligence/
│
├── README.md                 # Overview and entry point
├── STRUCTURE.md              # This file
├── LICENSE                   # MIT License
│
├── papers/                   # Theoretical papers
│   ├── abstraction_is_all_you_need.md      # Core APH framework
│   └── consciousness_emergent_abstraction.md # Consciousness application
│
├── notes/                    # Working notes and development
│   └── (future development notes, literature reviews)
│
└── experiments/              # Future empirical work
    └── (future experimental code and data)
```

## Directory Descriptions

### `/papers`
Formal theoretical papers with:
- Clear epistemic status labels (established, hypothesis, speculative)
- Testable predictions
- Falsification criteria
- Full references

### `/notes`
Working development materials:
- Literature review notes
- Theoretical development logs
- Draft ideas not yet formalized

### `/experiments`
Future empirical validation work:
- Code for testing predictions
- Data analysis
- Experimental protocols
